# Skills Test with GitHub Actions

Este proyecto contiene ejercicios prácticos del curso de GitHub Actions. Incluye pruebas automáticas definidas en Markdown y shell scripts personalizados.

## 📘 Contenido

- `.github/workflows` - Flujos de CI
- `.github/steps` - Pasos de prueba en Markdown
- `.github/script` - Scripts usados en workflows

## 🚀 CI/CD

Este proyecto está configurado para:
- Ejecutar scripts al hacer PR
- Validar estructura y reglas de pasos
- Aplicar acciones condicionales y matrices
